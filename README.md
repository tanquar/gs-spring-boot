# Setup

## Clone

```
$ git clone https://github.com/spring-guides/gs-spring-boot.git
```

## Move contents from ./complete

```
$ mv ./complete/src ./
$ mv ./complete/.mvn ./
$ mv ./complete/mvnw ./
$ mv ./complete/mvnw.cmd ./
$ mv ./complete/pom.xml ./
```

## Run

```
$ ./mvnw spring-boot:run
```

Visit http://localhost:8080/
